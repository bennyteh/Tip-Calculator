# Front-end Style Guide

## Layout

The designs were created to the following widths:

- Mobile: 375px
- Desktop: 1440px

## Colors

### Primary

- Strong cyan: hsl(172, 67%, 45%)

### Neutral

- Very dark cyan: hsl(183, 100%, 15%) (button-color/tip-card/)
- Dark grayish cyan: hsl(186, 14%, 43%) (text color title)
- Dark grayish cyan: hsl(184, 14%, 56%) (text color /person)
- Light grayish cyan: hsl(185, 41%, 84%) (background-color)
- Light grayish cyan: hsl(189, 41%, 97%) (input color)
- White: hsl(0, 0%, 100%) (calculator color)
-mint: hsl(172, 67%, 45%)

## Typography

### Body Copy

- Font size (form inputs): 24px

### Font

- Family: [Space Mono](https://fonts.google.com/specimen/Space+Mono)
- Weights: 400, 500, 700
